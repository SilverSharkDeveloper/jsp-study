디렉티브 태그
1.페이지 디렉티브 태그
<%@ page 속성명  = "속성값" %>
현재 페이지에 대한 정보를 설정하는 태그
되도록 페이지 최상단에 선언한다.

페이지 디렉티브 태그 속성
1.language(기본값: java) - 현재 JSP페이지가 사용할 프로그래밍 언어를 나타냄
2.contetnType(기본값: text/html) : 현재 JSP페이지가 생성할 문서의 콘텐츠 내용
3.pageEncoding(기본값:ISO-8859-1) : 현재 JSP페이지의 문자 인코딩 설정
4.import : 현재 JSP 페이지에서 사용할 자바 패키지 및 클래스 설정
5.session(기본값: true) - 해당 페이지에 세션 사용가능 여부
6.info : 현재 JSP 페이지에 대한 설명을 설정
7.errorPage - 해당 페이지에서 에러 발생시 보여줄 오류 페이지 설정 (경로설정)
8.isErrorPage(false) - 해당 페이지가 에러페이지인지 아닌지에 대한 여부


2.include 디렉티브 태그
	<% include file = "경로"%>
	
	현재 JSP페이지의 특정 영역에 외부 파일의 내용을 포함하는 태그이다.
	보통 header와 footer는 대부분의 페이지에서 동일한 내용으로 작성되기 때문에 	
	유지보수 및 편의를 위하여 외부 파일로 만든 후 include해서 사용한다.
	이러한 것을 모듈화라고 한다.

<액션 태그>
	서버나 클라이언트에게 어떤 행동을 하도록 명령하는 태그이다.
	페이지와 페이지 사이를 제어하거나 다른 페이지의 실행 결과 내용을 
	현재 페이지에 포함시킬 때 사용한다.
	또한 자바빈즈와 같은 다양한 기능을 제공하며 
	액션 태그는 XML형싁의 <jsp: />를 사용한다.

	1.forward(<jsp:forward page = ""/>)
		다른 페이지로의 이동, 페이지 흐름을 제어한다.

	2.include(<jsp:include page = ""/>)
		외부 페이지의 내용을 포함하거나 페이지 모듈화

	3.param(<jsp:param/>) ->forward태그 안에 작성
		현재 페이지에서 다른페이지에 정보를 전달한다.




input type -> button -> submit이 아니므로 action실행 x -> onclick 이벤트로 처리 
		(submit)하기전에 어떠한 처리가 필요한 경우 사용	
		->submit -> action 바로 실행
